De Definición

Crea una lista de definición donde definas los términos: HTML, W3C e Internet

HTML
Es el código que se utiliza para estructurar y desplegar una página web y sus contenidos.
W3C
Es el organismo mundial encargado de desarrollar las tecnologías y protocolos que han hecho posible la Web que hoy conocemos.
Internet
Red global de redes de ordenadores cuya finalidad es permitir el intercambio libre de información entre todos sus usuarios.
